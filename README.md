Online Quiz Console-Based Application 🎯

A Java-based console application that allows users to participate in an interactive online quiz. The application presents multiple-choice questions, evaluates answers, calculates scores, and displays the final result instantly.

This project was developed to strengthen concepts of Core Java, OOPs, Collections, and Exception Handling through a real-world mini project.

 Features
 Multiple-choice quiz system
 Interactive console-based interface
 Instant answer validation
 Score calculation and result display
 User-friendly navigation
 Clean and modular code structure
 
 Technologies Used
Java
OOPs Concepts
Collections Framework
Exception Handling
Scanner Class
Arrays / ArrayList
